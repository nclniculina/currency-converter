# Währungsrechner

Ein einfacher Währungsrechner, um Beträge zwischen verschiedenen Währungen zu konvertieren. 
Das Projekt zeigt grundlegende HTML-, CSS- und JavaScript-Fähigkeiten und eignet sich gut für ein Portfolio.

## Technologien 
- HTML
- CSS
- JavaScript

## Funktionen 
- Beträge eingeben und konvertieren
- Ergebnis sofort anzeigen
- Interaktive Buttons mit Hover-Effekten
- Dekorative Elemente zur besseren Benutzererfahrung


## Nutzung 
1. Alle Dateien (`index.html`, `style.css`, `script.js`) im selben Ordner speichern.  
2. `index.html` in einem Browser öffnen.  
3. Betrag und Währung auswählen, auf den Konvertieren-Button klicken.  
4. Das Ergebnis wird automatisch berechnet und angezeigt.

## Struktur des Projekts
Währungsrechner/
├─ index.html # HTML-Struktur der Seite
├─ style.css # Styling für Layout, Farben, Buttons
├─ script.js # JavaScript-Logik für Berechnungen
└─ README.md # Projektbeschreibung
